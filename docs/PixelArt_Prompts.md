# Mutating Gambit - 픽셀 아트 프롬프트

## 스타일 가이드

**선택된 스타일:**
- ✅ **16-bit 픽셀 아트** (SNES 스타일, 풍부한 색감)
- ✅ **명확한 실루엣** (작은 크기에서도 즉시 인식 가능)
- ✅ **투명 배경** (Unity 적용 용이)
- ✅ **미니멀 모던** (깔끔하고 현대적, 과도한 장식 없음)

## 5. 사용 가이드

### 추천 이미지 생성 도구
1. **Aseprite** (픽셀 아트 전문 도구)
2. **Lospec Pixel Editor** (무료 온라인)
3. **Stable Diffusion** (ControlNet + Pixel Art LoRA)
4. **Midjourney** (--style raw --q 2 파라미터 사용)

### 생성 시 주의사항
- **해상도 준수**: 정확히 64x64, 48x48, 32x32 픽셀로 생성
- **투명 배경**: PNG 포맷으로 배경 제거
- **안티앨리어싱 OFF**: 픽셀이 뭉개지지 않도록 설정
- **색상 제한**: 팔레트당 8-16색 이내로 유지 (16-bit 스타일)

### Unity 적용 방법
1. 생성된 이미지를 `Assets/Sprites/UI/` 폴더에 저장
2. Import Settings에서:
   - Texture Type: **Sprite (2D and UI)**
   - Pixels Per Unit: **1** (픽셀 퍼펙트)
   - Filter Mode: **Point (no filter)**
   - Compression: **None**
3. Artifact ScriptableObject의 `icon` 필드에 할당

## 6. 파라미터 제안

### Midjourney
```
--style raw --q 2 --no blur,3d,smooth --v 6
```

### Stable Diffusion
```
Sampling Steps: 20-30
CFG Scale: 7-9
Negative Prompt: (blur:1.3), (3d render:1.2), smooth, realistic, anti-aliasing
LoRA: pixel-art-xl or pixel-art-style (weight: 0.6-0.8)
```
