## 사용 가이드

1. **AI 이미지 생성 도구 선택**: Midjourney, DALL-E, Stable Diffusion 등
2. **프롬프트 복사**: 위의 프롬프트를 복사하여 사용
3. **커스터마이징**: 필요에 따라 색상, 크기, 스타일 조정
4. **배치**: Unity의 `Assets/Sprites/UI` 폴더에 저장
5. **참조**: Artifact ScriptableObject의 `icon` 필드에 할당

---

## 스타일 가이드

### 일관성 유지
- **색상 테마**: 다크 판타지, 중세 체스
- **질감**: 금속, 나무, 벨벳, 마법 에너지
- **효과**: 빛나는 효과, 파티클, 오라
- **해상도**: 64x64 (아이콘), 48x48 (태그), 32x32 (배지)

### 접근성
- 명확한 실루엣
- 높은 대비
- 색맹 고려 (형태로도 구분 가능)
- 작은 크기에서도 인식 가능
